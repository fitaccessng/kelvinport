# Kelvin Ibeh Portfolio

Modern founder portfolio built with React, Tailwind CSS, and Framer Motion.

## Development

```bash
npm install
npm run dev
```

## Backend (Contact API)

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r backend/requirements.txt
uvicorn backend.main:app --reload --port 8000
```
